# Pickid 웹 프로젝트 페이지별 PRD 목차

## 개요

이 문서는 Pickid 웹 프로젝트의 각 페이지별 Product Requirements Document(PRD) 목차를 정의합니다.

---

## 1. 홈 페이지 (Home Page)

**경로**: `/`  
**파일**: `src/app/page.tsx`

### 1.1 목적 및 개요

- 서비스 소개 및 진입점
- 인기 테스트, 추천 테스트, 카테고리별 테스트 노출

### 1.2 기능 요구사항

- [ ] 인기 테스트 섹션 (참여 수 기준 상위 6개)
- [ ] 추천 테스트 섹션 (최근 2주 내 생성된 테스트 우선, 없으면 시작 수 기준)
- [ ] 카테고리별 TOP 테스트 섹션 (완료율 기준 상위 6개)
- [ ] 카테고리 네비게이션
- [ ] 테스트 카드 클릭 시 상세 페이지 이동
- [ ] 반응형 디자인 (모바일 최적화)

### 1.3 데이터 요구사항

- [ ] 테스트 목록 (제목, 설명, 썸네일, 카테고리, 참여 수, 완료 수)
- [ ] 카테고리 목록
- [ ] 메타데이터 (동적 - 테스트 개수 포함)

### 1.4 UI/UX 요구사항

- [ ] 히어로 섹션
- [ ] 섹션별 제목 및 설명
- [ ] 테스트 카드 그리드 레이아웃
- [ ] 로딩 상태 처리
- [ ] 에러 상태 처리

### 1.5 성능 요구사항

- [ ] SSR로 초기 데이터 로드
- [ ] 이미지 최적화 (next/image)
- [ ] 메타데이터 동적 생성
